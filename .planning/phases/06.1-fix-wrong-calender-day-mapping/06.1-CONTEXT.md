## Phase Boundary

Fix incorrect day-of-week mapping in calendar event responses. Events are currently reported with the wrong weekday name (off by 1 day).

## Bug Description

**Current behavior:**

- User asks: "what do we do this weekend"
- Calendar has an event on Friday
- Bot responds saying the event is on Saturday (wrong day)

**Expected behavior:**

- Bot should correctly identify Friday events as Friday
- Day-of-week mapping should be accurate for all days

**Impact:**

- Users get incorrect information about when events occur
- Could lead to missed appointments or confusion
